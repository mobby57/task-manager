# Task Manager

A simple task manager web application built with HTML, CSS, and JavaScript.

## Features

- Add, edit, delete tasks
- Mark tasks as completed
- Filter tasks by status (all, pending, completed)
- Responsive design
- Local storage persistence
- Accessible UI

## AI Customization

This project includes custom instructions and chat modes for VS Code Copilot:

- `.github/copilot-instructions.md`: Coding guidelines
- `.github/chatmodes/Code Reviewer.md`: Code review mode

## Usage

Open `index.html` in a web browser to use the application.
